FizzBuzz: A Coding Project

Welcome to FizzBuzz! A Javascript app using loops, conditional statements, and template literals. Use the program that prints the numbers from 1 to a maximum number of your choice and multiples of two numbers of your choice. The first number prints "Fizz" and the second prints "Buzz". When the number is a multiple of both "FizzBuzz" is printed.


Instructions

Enter a fizz and a buzz value. Press the button and the values will appear.
